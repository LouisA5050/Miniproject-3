# Miniproject-3
Apple vs Samsung - Which brand performs better based on customer reviews?

This project analyzes 121,000+ smartphone reviews for Apple and Samsung to understand customer sentiment, compare ratings, and evaluate brand performance using machine learning. The work combines sentiment analysis, rating prediction, and classification models to identify patterns in customer feedback and determine which brand performs better based on reviews.  ￼

What Was Done
	•	Collected and analyzed a dataset containing Apple and Samsung smartphone reviews and ratings.
	•	Performed data cleaning and preprocessing to prepare text and rating data.
	•	Applied sentiment analysis to measure positive and negative customer opinions.
	•	Compared average ratings and positive sentiment rates between the two brands.
	•	Built machine learning models to predict review ratings and sentiment.
	•	Evaluated model performance using metrics such as MAE, RMSE, R², ROC-AUC, and confusion matrices.
	•	Created visualizations and presentation slides to communicate insights and business meaning.

Tools & Technologies Used
	•	Python
	•	Jupyter Notebook
	•	pandas / numpy – data manipulation
	•	scikit-learn – machine learning models (Logistic Regression, regression models)
	•	NLP techniques – sentiment analysis
	•	matplotlib / seaborn – data visualization

Opportunities for Improvement
	•	Improve text preprocessing (lemmatization, stop-word removal, advanced tokenization).
	•	Use advanced NLP models such as BERT or transformer-based sentiment models.
	•	Add feature engineering like review length, keyword frequency, or product category.
	•	Expand the dataset to include more smartphone brands and newer reviews.
	•	Implement hyperparameter tuning and cross-validation to optimize model performance.
	•	Build an interactive dashboard to explore brand sentiment and ratings in real time.

Overall, this project provides a strong foundation for consumer sentiment analysis and brand comparison using machine learning and NLP.
